Privacy Policy

Last updated: 2026-05-03

1. Overview

Layered Launcher ("the App") is a home launcher that runs on the user's Android device. The App does not transmit personal information to external servers, nor share it with any third party. All settings and usage history are stored solely on the user's device (local storage).

2. Information handled

The information the App handles on-device is listed below. None of it is transmitted off-device.

  • List of installed apps: read in order to lay out apps across floors
  • App launch history and counts: recorded for the "usage-driven floor change", "auto-move", and "app blocking" features
  • User settings: floor configuration, wallpaper, fonts, and other preferences
  • (Only when permitted) Notification metadata: source app and counts, used for notification batching
  • (Only when permitted) App usage statistics: read from the OS Usage API for the screen-time display

3. Storage

All of the above is stored only on the device using Hive (an on-device NoSQL store). No data is sent to any cloud, third-party server, or analytics service.

4. Third-party sharing

The App does not share collected information with any third party. No advertising, analytics, or crash-reporting SDKs are bundled.

5. Permissions

The Android permissions requested by the App are:

  • QUERY_ALL_PACKAGES: enumerate installed apps (core launcher functionality)
  • PACKAGE_USAGE_STATS (optional): screen-time display
  • BIND_NOTIFICATION_LISTENER_SERVICE (optional): notification batching
  • POST_NOTIFICATIONS: notifications for Mindful Delay and auto-move
  • READ_EXTERNAL_STORAGE / READ_MEDIA_IMAGES: wallpaper selection
  • SET_WALLPAPER: wallpaper application

Optional permissions only activate the related features when explicitly granted by the user. The App's core functionality works without them.

6. Data deletion

Uninstalling the App causes Android to remove all stored data automatically. You can also delete data from "Settings → Data Management" inside the App.

7. Changes to this policy

If this privacy policy is updated, the "Last updated" date on this page will change, and significant changes will be announced through in-app notifications.

8. Contact

For questions about this policy, please open an issue at GitHub Issues.
