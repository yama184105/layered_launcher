スマホとの距離を、
階層で設計する。

Layered Launcher は、アプリを「フロア（階層）」で整理する Android ホームランチャー。 アイコンを廃し、テキストのみ。日常のタップに、わずかな摩擦と意図を取り戻す。

β テスターになる 設計思想を読む

3 つの設計判断

1. アイコンを使わない

アイコンは無意識に手を伸ばさせる。テキストだけにすると、目で読み、判断してから触れることになる。 色彩刺激が消え、画面の情報量が減り、思考のための余白が戻る。

2. アプリは「階層 (フロア)」に置く

よく使うアプリは 1F。誘惑系は 5F。集中したいときに見えなくしたいものは 地下 B1F。 0F はホーム画面。フロアは 1F〜10F + 地下 B1F〜B10F まで設定可能。

3. 移動には小さな摩擦を入れる

フロア間の移動はアニメーションを介して行われる。マインドフルディレイで起動前にカウントダウンを置ける。 ストリクトモードで「変更前にタイマーを待つ」設定もできる。無意識のタップを意識のタップに変える仕掛け。

画面

スクリーンショットは順次差し替え予定。

機能

  • フロアシステム — 1F〜10F + 地下 B1F〜B10F + ホーム (0F)
  • 緊急モード — 指定アプリを 1F にも表示。日/週/年の使用回数制限つき
  • マインドフルディレイ — 起動前カウントダウン (3/5/10/30 秒)
  • 自動移動 — スケジュール (24h × 7 曜日) または日数間隔ランダム
  • 使用回数による階層変更 — 起動回数の閾値でフロアが上がる
  • ストリクトモード — 5 サブモードで設定変更にタイマー待ちを強制
  • 通知バッチ処理 — SNS 通知を指定間隔でまとめる
  • ジェスチャー — 上スワイプ・下スワイプ・ダブルタップに任意のアクション
  • アプリブロック — 時間帯・曜日・終日・使用時間上限の組み合わせ
  • 外観カスタマイズ — フォント・行間・時計フォーマット・壁紙・フロア背景色

なぜ作ったか

スマートフォンは「使われる」ものではなく、「設計するべき」ものになっている。 画面に並ぶアイコンは、メーカーや広告主の意図で配置された誘導の地図でしかない。

自分の生活を、自分でレイアウトし直したかった。 アプリに「フロア」を割り当て、必要な距離を意図的に置く。 それだけで、1 日の終わりに見るスクリーンタイムが変わる。

β テスター募集

Play Store のクローズドテストに参加してくれる方を募集中。14 日間、Galaxy など Android スマートフォンで毎日数分でも開いてくれれば OK。

テスターに求めること

  • Android スマートフォン (Android 10 以降推奨) を使っていること
  • Google アカウント (Gmail) を持っていること
  • 14 日間、たまにアプリを開いてくれること
  • 不具合や違和感があれば、簡単なメモで知らせてくれること (任意)

よくある質問

テスト期間中、何をすればいい？

毎日数分 Layered Launcher をホームとして使うだけで OK。詳細なフィードバックは任意。

普段使っているランチャーに戻せる？

はい。Android の設定からデフォルトホームをいつでも変更できる。アンインストールも自由。

有料？

β 期間中は無料。正式公開後の価格は未定 (現時点では無料配布の方針)。

iPhone 版は？

未対応。iOS のホームランチャー API はサードパーティに開放されていないため、原理的に作成不可。

個人情報は集まる？

クラウドへのデータ送信は行わない。すべてのデータはローカル (Hive) に保存。詳細はプライバシーポリシー。
