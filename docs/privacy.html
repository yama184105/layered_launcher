プライバシーポリシー

最終更新日: 2026-04-28

1. 概要

Layered Launcher (以下「本アプリ」) は、ユーザーの Android スマートフォン上で動作するホームランチャーです。 本アプリは、ユーザーの個人情報を外部サーバーに送信したり、第三者と共有したりすることはありません。 すべての設定および利用履歴はユーザーの端末内 (ローカルストレージ) のみに保存されます。

2. 収集する情報

本アプリが端末内で扱う情報は以下のとおりです。これらはいずれも端末から外部に送信されません。

  • インストール済みアプリの一覧: フロアへの配置・表示順制御のために参照します
  • アプリの起動履歴・回数: 「使用回数による階層変更」「自動移動」「アプリブロック」機能のために記録します
  • 各種設定値: フロア構成・壁紙・フォント等のユーザー設定
  • (端末設定で許可した場合のみ) 通知内容のメタ情報: 通知バッチ処理機能のために、通知の送信元アプリ・件数を読み取ります
  • (端末設定で許可した場合のみ) アプリ使用時間統計: スクリーンタイム表示のために OS の使用時間 API を参照します

3. 保存場所

上記すべての情報は、Hive (オンデバイス NoSQL ストレージ) を用いて端末内のみに保存されます。 クラウド・第三者サーバー・解析サービスへの送信は一切行いません。

4. 第三者への提供

本アプリは収集情報を第三者に提供しません。広告 SDK、解析 SDK、クラッシュレポート SDK 等の外部 SDK を一切組み込んでいません。

5. 権限

本アプリが要求する Android の権限は以下のとおりです。

  • QUERY_ALL_PACKAGES: インストール済みアプリ一覧の取得 (ランチャーのコア機能)
  • PACKAGE_USAGE_STATS (任意): スクリーンタイム表示
  • BIND_NOTIFICATION_LISTENER_SERVICE (任意): 通知バッチ処理
  • POST_NOTIFICATIONS: マインドフルディレイ・自動移動の通知
  • READ_EXTERNAL_STORAGE / READ_MEDIA_IMAGES: 壁紙の選択
  • SET_WALLPAPER: 壁紙の設定

任意権限はユーザーが明示的に許可した場合のみ機能します。許可しなくても本アプリのコア機能は動作します。

6. データの削除

本アプリをアンインストールすると、保存された全データは Android OS により自動的に削除されます。 アプリ内の「設定 → データ管理」からも個別に削除できます。

7. 本ポリシーの変更

本プライバシーポリシーが更新された場合、本ページの「最終更新日」を変更し、 重要な変更がある場合はアプリ内通知でお知らせします。

8. 連絡先

本ポリシーに関するお問い合わせは、 GitHub Issues までお願いします。
